# Shared ToDoLists

Веб-додаток для створення, редагування та публічного перегляду списків задач.

## Встановлення

1. Встановіть залежності:
```bash
pip install -r requirements.txt
```

2. Запустіть додаток:
```bash
python flask_app.py
```

3. Відкрийте в браузері:
```
http://localhost:5000
```

## Використання

- **Сторінка редагування** (`/`): Створення та редагування списків задач
- **Сторінка перегляду** (`/view/<list_id>`): Публічний перегляд списку задач

## Технології

- Flask (Python) - бекенд
- SQLite - база даних
- Alpine.js - реактивність
- Bulma - стилізація
